# Shared Setup — Convergence Steps

Both **Skillable** and **Self-Guided** paths converge here. Whether you
ran `azd up` yourself or your Skillable environment was pre-provisioned,
the rest of setup is identical.

## Steps

| Step | Action |
|------|--------|
| [1](./1-discover-env.md) | Run `scripts/discover-env.sh` to populate `.env` from the live Azure resources |
| [2](./2-three-tabs.md) | Open the three browser tabs and confirm readiness for Lab 1 |

After these two steps, you'll be ready to start [Lab 1](../../core/lab-1.md).
